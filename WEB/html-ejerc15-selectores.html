Tipos de selectores que entran en el examen

Selector: Clase

El selector clase nos permite aplicar un estilo poniendo class="nombre", y el estilo se aplicará a todos los elementos que tengan ese nombre en su clase como en este caso que pondré este párrafo en rosa gracias a la clase.

Selector: Identificador

El selector id nos permite aplicar un estilo a todos los elementos que tengan (id="algo") el mismo id en este caso lo aplicaré en este párrafo con el color verde.

Selector: Todos los elementos

El selector todos los elementos(*) se aplica al principio de la página y aplica un estilo a TODOS los elementos de la página pero si luego se aplica otro estilo ,por ejemplo a un párrafo, el último prevalece sobre el "*" (todos los elementos).
El ejemplo que he puesto para este caso es que aplique al principio para que todos los elementos tengan margen y relleno entre ellos de 5 píxeles.

Selectores varios: p , em , h1 , h2, ...

Sirve para poniendo varios elementos entre comas se les apilcaráun mismo estilo a todos ellos el el ejemplo es el color del fondo verde de los elementos entre las comas que en este caso son los elementos h1 y em

Selectores contenidos en otros selectores: ol li

Se utilizan para aplicarle un estilo a un elemento que esté dentro de otro y el ejemplo es la lista ordenada de a continuación.

  1. Mañana
  2. Tarde
  3. Noche

Selector: Sobrevuleo (p:hover)

Se utiliza para que un elemento que hayamos seleccionado al pasar el cursior por encima de eél cambie su estilo al que elijamos,en éste caso el ejemplo es esta oración en negrita que al pasar el cursor su fondo cambiará a azul.
